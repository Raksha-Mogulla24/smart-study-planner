# Smart Study Planner

A Python project that helps students plan and manage their study schedule efficiently.

## Features
- Add study tasks
- Track completed tasks
- Store study data using CSV files
- Simple command-line interface

## Technologies Used
- Python
- CSV file handling

## How to Run the Project

1. Install Python
2. Download the project files
3. Run the program

python smartstudyplanner.py

## Project Structure
smartstudyplanner.py  - Main program
study_data.csv        - Stores study tasks
